# 鉄道の乗りつぶしが趣味です
東海地方はもちろん、なんと四国のJRを制覇している友人がいてそれに触発されて鉄道の乗りつぶしをやっています  
去年は東京メトロ、都営地下鉄、京王、東急、相鉄と関東の会社を大量に制覇したのですが、なんと地元の会社である名鉄は最後に乗りつぶししてから2年くらい放置していたんですよね  
本当はシングレコラボの時にやりたかったけど時間がなかった…  
とはいえ今も時間がない事には変わりないですが、もう愛知を出ていくまで時間がない！今やります

# さあ乗りつぶしだ

人生で２番目に使ってる駅が刈谷のはずなのに、なぜかここは一度も乗ったことないんですよね～  
厳密に言うと酔った勢いで寝過ごしたことはあるんですが、それはノーカンで今回が初乗車  
フリー切符を買いまして、刈谷駅の始発でいざ碧南へ  
にしても酔いどれだらけで治安悪い…

なにか言うとしたら閉館したはずの刈谷日劇が入っているビルの電気が付いていたくらいか  
6:32に碧南に到着です  
特に何もすることはないのですぐ折り返します

そして車内には共通テストを受けに行くだろう生徒がたくさん…頑張れ  

## 西尾線(西尾～吉良吉田)
新安城～西尾は西尾線の一部特別車特急がなくなるときに乗ったことがある故にここだけ残っています  
さっそく知立の悪名高い乗り換えを終えて特急に乗りこみ、新安城で下車  
やけに人が多い。コンコースで「頑張ってくださ～い」って言ってる人いるしここら辺にも会場あるのか？と思ったら  
<span class="text-xl">まさかの全員マラソン大会のランナーでした</span>  
西尾でマラソン大会があるみたいです。  

特急の乗り換え先の列車は西尾止まりなので、やりたかった別件の消化もかねて南安城で一度下車。  
(碧南～南安城 690円 計1,150円)  
こういう地方の駅前っていいな～。　そしてやたらと夜のお店が多い。  
一本後の列車で吉良吉田まで行き、これで西尾線制覇です！(特に言うことがない)

# 広見線
なんでこんな極端な行程にしたんだろうと2年前の自分に対して思いますが、考え直すのも面倒なのでそのままやります  
まずは犬山まで行かなければなりません  
が、先ほど紹介したように蒲郡線を使って行くと名古屋まで100分かかる体たらくなので、東海道線でショートカットをします  
しかし蒲郡～名古屋は1,000円とさすがにもったいないので、豊橋まで行ってそこから快速特急でワープします  
<div class="mx-auto flex flex-col items-center justify-center ">
<img src="/posts/meitetsu/toyohashi.jpg"/>
結局特別車課金するんですけどね
</div>

どうやら国府駅で停電が起きていたようで、若干遅れて入線しました  
運転見合わせにならなかったのはラッキーだけどちょっと旅程崩壊が怖い  
結局犬山には5分遅れで到着、本来は3分乗り換えだったので旅程崩壊…のはずでしたが、接続待ちをしてくれたのでそれは逃れました　感謝  

早速広見線に乗車したのですが犬山を出てからいきなり田園風景が広がります  
これは新可児から先どうなるんだろうか…
というかなぜか複線なんですよね　主に朝のミュースカイのため…??  
新可児に近づくとさすがに市街地の形相を示します
<div class="mx-auto flex flex-col items-center justify-center ">
<img src="/posts/meitetsu/shinkani.jpg"/>
新可児に到着
</div>

新可児で御嵩行きの列車に乗り換えます　同じ広見線ですが完全に分断されていて、中間改札を通る形になっています  
新可児から先はより田園風景が強くなるのですが、やはり家が全くない…ってことはないんですよね  
蒲郡線同様車の方が便利な故見放されている感じがありました  
名鉄は可児・御嵩の観光切符を売っているようで、それを利用して旅行中の家族が二世帯いました
<div class="mx-auto flex flex-col items-center justify-center ">
<img src="/posts/meitetsu/mitake.jpg"/>
広見線制覇 (豊橋～御嵩 1,950円 計4,000円)
</div>

午前中にフリー切符の元が取れましたね  
ここまでずっと列車に乗り続けていたので、ここで30分だけ観光をしてみました  
どうやら中山道の宿場町だったみたいですね　以前馬籠宿を訪れたことがあるので訪問は二か所目となります  
カフェに入ろうとしたのですがなんと予約で満席になっており入れず…  
代わりに駅前にある商店で御嵩土産を買ってみました　地元の人たちが作ったキャラメル…？らしい

ここからは小牧線に乗車していきます  
ちなみにここでも別件のため小牧原駅で下車しました (御嵩～小牧原 750円 計4,750円)  
正直よくある市街地から外れた国道沿いでなにもコメントすることはありません()  
ただこの駅、ホームの真上をある構造物が通っています

# 瀬戸線
瀬戸線は名鉄で唯一他の路線と接続していません  
なので他社線を使ってどうにかたどり着く必要があります  
上飯田からそのまま電車に乗り続けると平安通に到着します  
平安通で名城線に乗り換え、一駅行くと名鉄瀬戸線が乗り入れる大曽根駅に到着です  

まずは栄町をほったらかし尾張瀬戸の方を制覇します  
<span class="text-xl">瀬戸線、おもしれえ！！</span>  
名古屋市内はとにかくアトラクション感がすごい  
住宅街を無理やり縫って作られたような構造ですよ  
瓢箪山のホームとかどうなってんですかねあれ  

その一方で小幡からはなんか同じような光景がずっと続く路線で正直…でした  
でも名古屋の中心部から離れても似たような栄え方が続くってなかなかですよね  
私、尾張旭は初通過で瀬戸自体も２度目とかなんですが、瀬戸、侮ること勿れって感じです  
そんなこんなで尾張瀬戸に到着しました  
(大曽根～尾張瀬戸 460円 計5,610円)
<div class="mx-auto flex flex-col items-center justify-center ">
<img src="/posts/meitetsu/owariseto.jpg"/>
尾張瀬戸では乗ってきた電車が故障で交換になるトラブルが
</div>

そのまま引き返して栄町に向かいます  
大曽根から続く高架と東大手で地下に潜るところを楽しみにしていたのですが、さすがにここまで電車に乗り続けた疲れは大きく寝てしまいました…  
というわけであっという間に栄町到着です  
(尾張瀬戸～栄町 510円 計6,120円)
<div class="mx-auto flex flex-col items-center justify-center ">
<img src="/posts/meitetsu/sakaemachi.jpg"/>
瀬戸線制覇！
</div>

旅程に少し余裕があったのでテレビ塔を下から見上げてから次の予定に移ります  
ザ・ランドマーク栄の開業も楽しみですね～

# 築港線・名古屋鉄道完全制覇
前述のとおり瀬戸線は孤立しているのでまた他社線を使って移動する必要があります  
ここでは名城線の栄～金山を利用して移動しました  
金山駅からは常滑線に乗車、大江で下車します  
<span class="text-3xl">で、でた！！！</span>  
<div class="mx-auto flex flex-col items-center justify-center ">

東名古屋港駅の改札

大江駅の中ですが大江駅ではありません、東名古屋港駅です  
東名古屋港駅には改札がないので大江駅のこの改札がその役割を果たします  
まあ蒲郡線や広見線でも似たようなことをやっているので今日3回目ですね

築港線の名所としては東名古屋港駅の手前にあるダイヤモンドクロッシングです  
ダイヤモンドクロッシングってなんやねん！！って感じですが簡単に言うと線路同士が直角に交差します  
その時のジョイント音が独特で面白いのでぜひYouTubeとかで調べてみてください  

またこの路線を最後にした理由として、通勤需要がメインの路線で朝夕以外に列車が運転されていないことがあります  
この乗車も大江→東名古屋港の夕方最初の運行だったので、(しかも休日)車内には鉄オタのキッズしかいませんでした
